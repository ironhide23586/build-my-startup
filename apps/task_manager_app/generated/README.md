# Task Management CLI Tool Documentation

## Overview
This CLI tool allows users to manage tasks through a command-line interface. Users can add tasks, mark them as complete, list all tasks, and delete tasks. Task data is stored in a JSON file.

## Installation

1. Ensure you have Python 3.x installed on your system. You can download it from [python.org](https://www.python.org/downloads/).

2. Install any required dependencies (if applicable). For this tool, you can ensure your Python environment is up to date.
